# 📱 Mobile Phone Addiction Prediction (Machine Learning Project)

## 🔍 Problem Statement

With the rapid increase in smartphone usage, many users are experiencing addiction-related issues that affect their daily life, productivity, and mental health.
This project aims to predict whether a user is addicted to mobile phone usage based on behavioral and usage patterns using Machine Learning.

---

## 🎯 Objective

* Analyze smartphone usage data
* Build a predictive Machine Learning model
* Classify users as **Addicted** or **Not Addicted**
* Deploy an interactive web application for real-time prediction

---

## 🧠 Machine Learning Model

* Model Used: **Logistic Regression**
* Type: Classification
* Evaluation Metrics:

  * Accuracy Score
  * Precision, Recall, F1-Score

---

## ⚙️ Technologies & Tools

* Python 🐍

## 🧪 Techniques Used

* Data Cleaning & Preprocessing
* Handling Missing Values (Mean & Mode Imputation)
* Feature Engineering
* One-Hot Encoding (Categorical Data)
* Standard Scaling (Numerical Data)
* Pipeline & ColumnTransformer
* Model Training & Evaluation

---

## 📊 Features Used

* Age
* Gender
* Daily Screen Time
* Social Media Usage
* Gaming Hours
* Work/Study Hours
* Sleep Hours
* Weekend Screen Time
* Stress Level
* Academic/Work Impact
* Addiction Level

---

## 🚀 Web Application

An interactive web application is built using Streamlit where users can:

* Input their daily smartphone usage data
* Get instant prediction (Addicted / Not Addicted)
